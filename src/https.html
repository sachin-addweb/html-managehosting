Helpful Hints

  • HTTPS

    Turning this on will secure your site by configuring Nginx to serve traffic over HTTPS. Regular HTTP traffic will be redirected to HTTPS. HTTP/2 will also be configured, which will improve the performance of your site.

    You can choose between a free Let's Encrypt certificate, which will be generated and then renewed every 45 days automatically. Or, install a custom certificate, which must be uploaded to the server and renewed manually.

    We recommend that all sites use HTTPS. Not only will it improve the security of your site, but it can also help with SEO since HTTPS is used as a ranking signal. Note that some browsers will even flag sites as Not secure if they dont use HTTPS.
